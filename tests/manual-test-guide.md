# 手动测试指南

## 测试环境准备

1. 确保开发服务器正在运行：`npm run dev`
2. 访问：http://localhost:3002

## 测试步骤

### 1. 创建新日记

1. 点击"写日记"按钮
2. 输入标题：`测试 Markdown 渲染`
3. 在编辑器中输入以下内容：

```markdown
# Markdown 渲染测试

## 文本格式

这是**粗体文本**和*斜体文本*，以及~~删除线~~。

## 有序列表

1. 普通第一项
5. 自定义序号第五项
10. 大序号第十项

## 无序列表

- 项目 1
- 项目 2
  - 子项目 2.1
  - 子项目 2.2

## 任务列表

- [x] 已完成的任务
- [ ] 未完成的任务

## 代码块

### 行内代码
这里有 `console.log()` 代码。

### JavaScript 代码
```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome, ${name}`;
}

greet("World");
```

## 引用

> 这是一段引用
> 可以有多行
>> 嵌套引用

## 表格

| 功能 | 支持情况 | 备注 |
|------|---------|------|
| 标题 | ✅ | 支持 H1-H6 |
| 列表 | ✅ | 有序和无序 |
| 代码 | ✅ | 语法高亮 |
| 表格 | ✅ | 完整支持 |

## 链接和图片

[GitHub](https://github.com)

## 分割线

---

测试完成！
```

4. 点击"保存"按钮

### 2. 验证渲染效果

在日记详情页检查：

- ✅ 标题 H1、H2 正确渲染
- ✅ 粗体、斜体、删除线显示正确
- ✅ 有序列表显示：1.、5.、10.
- ✅ 无序列表和子列表缩进正确
- ✅ 任务列表显示复选框
- ✅ 代码块有语法高亮
- ✅ 引用有左边框装饰
- ✅ 表格格式正确
- ✅ 链接可点击
- ✅ 分割线显示

### 3. 测试编辑器功能

1. 返回并编辑刚创建的日记
2. 测试快速插入按钮：
   - 点击"标题" - 应插入 `## 标题`
   - 点击"列表" - 应插入无序列表
   - 点击"有序" - 应插入有序列表
   - 点击"引用" - 应插入引用语法
   - 点击"代码" - 应插入代码块
   - 点击"链接" - 应插入链接语法
   - 点击"表格" - 应插入表格
3. 测试预览模式：
   - 点击"预览模式"按钮
   - 应该看到渲染后的效果
   - 点击"编辑模式"返回编辑

### 4. 测试图片上传

1. 拖拽一张图片到编辑器
2. 粘贴一张图片（Ctrl+V）
3. 点击"📷"按钮选择图片
4. 保存并验证图片显示

## 预期结果

- 所有 Markdown 元素正确渲染
- 编辑器和显示页效果一致
- 图片正确显示
- 预览模式工作正常
- 快速插入按钮功能正常
