新增命令：
---------
- npm run version - 查看当前版本
- npm run docker:build - 构建带版本标签的Docker镜像
- npm run docker:tag - 为Docker镜像打latest标签
- npm run docker:push - 推送镜像到Docker Hub
- npm run release - 发布新版本（自动更新版本号、构建和推送）

API端点：
--------
- /api/health - 健康检查，包含版本信息
- /api/version - 详细的版本信息查询

Docker镜像标签：
--------------
- leowinfy/10mins:v4.1.0 - 具体版本
- leowinfy/10mins:latest - 最新版本

注意事项：
--------
1. 版本号遵循语义化版本控制（Semantic Versioning）
2. 发布新版本时使用npm version命令自动更新
3. Docker构建时会自动包含版本信息
4. 所有组件都可以获取到当前版本信息
